Create beautiful, vivid documents

Publications includes all the tools to create beautiful documents with shapes and text. Shapes can be customized to have different fill colors, border colors, border width, radii, opacity and size. Publications now supports canvas zooming and object snap-to-grid alignment. Export your documents to scalable multi-layer PDFs that can be shared with anyone.

Sort, search and save documents

Publications saves all your documents securely on the cloud. Easily view all your documents with full crisp and clear previews. Publications now lets you search and filter documents by name or date.

Full-stack JavaScript web app

Publications is built using full-stack JavaScript technologies running on an ArchLinux cloud-based server. There are no additional plug-ins or downloads required to run Publications. Publications runs on the latest versions of Safari, Chrome and Opera for Mac and Windows.

Publications is designed and developed by Michael Kelly and Carlos Paelinck.
