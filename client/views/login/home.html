Create beautiful, vivid documents

Publications includes all the tools to create beautiful documents with shapes and text. Shapes can be customized to have different fill colors, border colors, border width, radii, opacity and size. Publications now supports canvas zooming and object snap-to-grid alignment.

Cloud-based web app

Publications saves all your documents securely on the cloud. Publications now lets you search documents by name or date. Export your documents to scalable multi-layer PDFs that can be shared with anyone.

Accelerated Performance

Publications is a full-stack JavaScript app built with NodeJS and AngularJS. There are no downloads or plug-ins to install to use Publications. It uses SVG-based graphics which look crisp on both standard and HiDPI displays.

Publications developed and designed by Michael Kelly and Carlos Paelinck.
