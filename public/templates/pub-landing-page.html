Publications

Page layout & screen design web app

Publications is engineered and designed as a high-performance web app for beautiful page layout and screen design.

Publications uses a SVG-based drawing engine. Everything is a vector and looks crisp and sharp from small displays to retina displays.

Create shapes with vivid color, transparency, rounded rectangle corners and borders of various sizes.

Create colorful documents and layouts using a responsive, fast user interface.

A contextual inspector only shows the tools and settings for the selected object. Shapes can be colorized using a color picker with 21 vibrant swatches or inputted using HEX values.

Publications exports documents to lightweight multi-layer PDF documents with full-vector shapes. These PDF documents print beautifully at any size.

Publications works on Macs, PCs or iPads with the latest versions of Safari, Chrome, Firefox or Internet Explorer.

Publications is a full-stack JavaScript web application operating on the strengh of open-source frameworks and UNIX.

Publications is an object-oriented web app using BackboneJS, NodeJS and d3 frameworks running on an ArchLinux server.

Design and Development is an ongoing collaboration and effort by Michael Kelly and Carlos Paelinck.
