Publications. Engineered and designed as a high-performance web app for beautiful page layout and screen design.

Visit Publications on your PC or Mac to start creating documents.

High-Performance Web App

Publications is up to 21× faster¹ than the previous generation and is 7.5× smaller². It works with the latest versions of Safari, Chrome, Firefox and Internet Explorer³.

Full-Stack JavaScript Architecture

Built upon the Node, Express and Backbone JavaScript framworks for a high-performance, plug-in free experience.

SVG Drawing Engine

Publications uses the SVG-based d3 drawing library. Everything is a vector and looks crisp and sharp from small displays to HiDPI Retina displays.

Stroke, Transparency and Color

Publications now supports color transparency, rounded rectangle corners and shape borders of various colors and sizes.

Text and Typefaces

Add beautiful, crisp text to your documents with support for various typefaces. Coming Soon.

Multi-Layer Vector PDFs

Publications exports documents to lightweight multi-layer PDF documents with vector shapes, lines and text.
