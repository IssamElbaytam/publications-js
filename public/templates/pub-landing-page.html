High-Performance Web App

Publications is up to 21× faster¹ than the previous generation and is 7.5× smaller². It works with the latest versions of Safari, Chrome, Firefox and Opera Next.

Full-Stack JavaScript Architecture

New

Built upon the Node, Express and Backbone JavaScript framworks for a high-performance, plug-in free experience.

SVG Drawing Engine

New

Publications uses the SVB-based d3 drawing library. Everything is a vector and looks crisp and sharp from small displays to retina displays.

Stroke, Transparency and Color

New

Publications now supports color and stroke transparency, rounded rectangle corners, shape borders and HEX colors.
